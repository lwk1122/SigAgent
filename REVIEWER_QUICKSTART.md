# Reviewer Quickstart

This quickstart verifies the core SigAgent decision pipeline without requiring vendored MuSiCal, AMuSa, SigProfilerAssignment, or unpublished/local assets.

For the release boundary, inclusion/exclusion policy, and current gate status, see `RELEASE_PACKAGE.md`.

## Scope

The release smoke suite uses deterministic toy SBS96 fixtures generated by `experiments/make_release_smoke_fixtures.py`. These fixtures are for installation and pipeline checks only. They are not biological evidence and should not be used as manuscript performance results.

## Setup

```bash
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

Optional full local wrappers may require additional dependencies:

```bash
pip install -r requirements-optional.txt
```

## Core Checks

Run unit tests:

```bash
python -m unittest discover -s tests
```

Run the release smoke suite:

```bash
python experiments/run_paper_suite.py experiments/configs/paper_release_smoke.json
```

Expected outputs:

- `results/paper/paper_release_smoke/fixtures/manifest.json`
- `results/paper/paper_release_smoke/manifests/suite.manifest.json`
- `results/paper/paper_release_smoke/raw/known_sbs96_toy_plain_nnls/aggregate_metrics.tsv`
- `results/paper/paper_release_smoke/raw/known_sbs96_toy_plain_nnls/per_sample_metrics.tsv`

The fixture manifest records SHA256 checksums for generated sample, signature, and exposure CSV files.

Local verification on 2026-05-14:

## Paper Artifact Checks

If the full paper-result artifact directory is included, validate expected manuscript outputs:

```bash
python experiments/validate_paper_outputs.py --root results/paper --output-dir results/paper/paper_readiness
python experiments/make_paper_figures.py --root results/paper --output-dir paper/figures
python experiments/validate_release_readiness.py --repo-root . --results-root results/paper --output-dir results/paper/reviewer_package_readiness --release-stage review_package
```

The review-response SBS96 suite used by the current Figures 2-4 can be regenerated with:

```bash
python experiments/run_paper_suite.py experiments/configs/paper_review_response_sbs96.json
python experiments/collect_results.py results/paper/paper_review_response_sbs96
python experiments/make_tables.py results/paper/paper_review_response_sbs96
python experiments/make_calibration_tables.py results/paper/paper_review_response_sbs96 --assignment-f1-threshold 0.3
python experiments/summarize_catalog_assessor.py results/paper/paper_review_response_sbs96/artifacts/catalog_assessor_sbs96_v2.json --output-dir results/paper/paper_review_response_sbs96/tables
python experiments/make_review_response_summary.py results/paper/paper_review_response_sbs96
```
